# Melbourne & Sydney Winter Trip 2026

A single-page React + Tailwind CSS travel plan for the 2026 Australia winter trip.

## Run locally

```bash
npm install
npm run dev
```

## Edit trip content

Most editable travel content lives in:

```text
src/data/tripData.js
```

Main sections are split into:

```text
src/components/Hero.jsx
src/components/Timeline.jsx
src/components/TravelGuide.jsx
src/components/Navbar.jsx
```
